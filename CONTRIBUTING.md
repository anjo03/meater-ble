# Contributing

Bug reports, hardware test reports, and focused pull requests are welcome.

Include Home Assistant version, MEATER BLE version, probe model, Bluetooth adapter or proxy details, diagnostics, and relevant logs. Never publish credentials, Wi-Fi passwords, API keys, precise addresses, or unredacted private diagnostics.

Update documentation and translations for user-facing changes, update `CHANGELOG.md`, and ensure HACS, Hassfest, Python compilation, and JSON validation pass.

Versioning examples: `0.9.0b1`, `0.9.0b2`, `1.0.0`. Git tags use a `v` prefix and must match `manifest.json`.
